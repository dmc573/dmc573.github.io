1.2.3 Creating Mobile Apps Questions

Questions:
1. App Inventor is similar to Scratch in that they both have click-and-drag programming to assemble the code. However, Scratch allows you to program computer web games while App Inventor programs mobile apps instead. Therefore Scratch and App Inventor both have different user inputs and outputs to use in programs.
2. There are different inputs that a program can implement on a phone, like the microphone, touch screen, accelerometer, etc. while a computer relies on mouse and keyboard input for most programs.
